Hari Ini Kita Belajar... 🚀

🏗️ 1. Persiapan Latar

Menyiapkan Backdrop dan memposisikan Sprite Pemain & Bola.

🏃‍♂️ 2. Program Pemain

Membuat pemain berlari ke arah bola menggunakan blok Glide dan ganti kostum.

⚽ 3. Program Bola

Mengatur timing bola agar meluncur tepat saat 'ditendang' oleh pemain.

🎵 4. Program Musik

Menambahkan musik latar agar suasana pertandingan semakin seru!

Studi Kasus: Football Animation 🌟

Apa yang terjadi di sini?

Proyek ini menggabungkan:
⚡ Event: Klik Bendera Hijau.
🚶 Motion: Bergerak & Meluncur (Glide).
👕 Looks: Ganti Kostum saat menendang.
🔊 Sound: Peluit & Musik Latar.

👉 BUKA SCRATCH DI SINI 👈

Kalau sudah pede kamu bisa mulai kerjakan sendiri,
tapi kalau masih bingung simak layar dan setelah ini kita akan kerjakan sama-sama!

1. Mempersiapkan Latar & Sprite 🏟️

Pilih Backdrop: Soccer
Tambah Sprite: Jordyn
Tambah Sprite: Soccer Ball

Siapkan pemain di kiri dan bola di depannya!

2. Program di Sprite Pemain 🏃‍♂️

when green flag clicked
go to x: -167 y: -77
start sound (Refree Whistle)
switch costume to (jordyn-a)
glide 1 secs to x: -68 y: -80
switch costume to (jordyn-b)

3. Program di Sprite Bola ⚽

when green flag clicked
go to x: 0 y: -87
wait 1 seconds
glide 1 secs to x: -63 y: 52

4. Program pada Backdrop 🎵

when green flag clicked
play sound (Drum Satellite) until done
